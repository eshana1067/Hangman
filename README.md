# Hangman Game 🎮

A simple Hangman game built using Python. The player tries to guess a hidden word by entering one letter at a time. The player has limited attempts to guess the word correctly.

## 📌 Features
- Random word selection
- User input for guessing letters
- Limited number of attempts
- Win and lose conditions
- Simple and beginner-friendly Python code

## 🛠️ Technologies Used
- Python 3
- Random module
